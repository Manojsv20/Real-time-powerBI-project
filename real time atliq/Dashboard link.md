DASHBOARD 360



Project Folder – Complete Resources

Link:[ https://drive.google.com/drive/folders/13zhOzPXjNvsJ-bceR\_Tv7L3MTYJvYktl?usp=sharing](https://drive.google.com/drive/folders/13zhOzPXjNvsJ-bceR_Tv7L3MTYJvYktl?usp=sharing)



This folder contains the entire set of project materials used in Dashboard 360, including raw datasets, processed data files, model reference files, documentation, Excel sources, PBIX support materials, and all intermediate resources generated during the data-cleaning and analytical stages. It serves as the main repository of every file required to understand, rebuild, or verify the project.



Final Power BI Dashboard (PBIX File)

Link: [https://drive.google.com/file/d/1GBe3oN66kJVOempENV2jDw4XkOwwk77g/view?usp=sharing](https://drive.google.com/file/d/1GBe3oN66kJVOempENV2jDw4XkOwwk77g/view?usp=sharing)



This PBIX file represents the completed Dashboard 360 model, containing the final data structure, relationship mapping, DAX calculations, KPIs, visuals, report themes, slicers, and page designs. It is the final deliverable that showcases all analytical insights produced in the project.



Dashboard Demonstration Video

Unzipped Database Files – gdb041
